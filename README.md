# 🩺 Diabetes Prediction System

A Machine Learning web application built using FastAPI and Streamlit to predict diabetes.

## Features

- Diabetes prediction using a trained Logistic Regression model
- FastAPI backend
- Streamlit frontend
- Input validation with Pydantic
- Confidence score for predictions

## Tech Stack

- Python
- Scikit-learn
- FastAPI
- Streamlit
- NumPy
- Pandas
- Joblib
